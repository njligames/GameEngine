:Title:
MultipleRenderTargets
:ShortDescription:
Shows how to use multiple render targets to output data to several different surfaces within a single pass. 

:FullDescription:
Multiple Render Targets is a technique utilised in modern graphics processing that allows the rendering of multiple images to multiple render targets simultaneously. These textures can then be used as texture maps applied to 3D models, or as inputs for other shaders.
